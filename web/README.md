# web

The Next.js 16 app for zaffa. See the root [`README.md`](../README.md) and [`CLAUDE.md`](../CLAUDE.md) for the big picture.

## Commands

```bash
bun install
bun dev          # dev server
bun run build    # type-check + production build — the verification gate
bun run lint     # eslint
bun run typecheck
```

**Next.js 16** has breaking changes from earlier versions. Before writing Next.js-specific code, read the docs in `node_modules/next/dist/docs/` (see [`AGENTS.md`](./AGENTS.md)).
